# ☕ Brew & Shots – Luxury Coffee Bar Website

A modern, premium multi-page coffee bar and restaurant website designed to deliver a luxury dining experience. The project features an elegant UI, responsive layouts, an interactive menu, shopping cart functionality, and Google Sheets backend integration for customer enquiries and bookings.

---

## ✨ Features

- Premium Luxury UI/UX Design
- Fully Responsive Layout
- Multi-Page Website
- Interactive Food & Beverage Menu
- Product Search & Category Filters
- Shopping Cart System
- Dynamic Product Rendering
- Google Apps Script & Google Sheets Integration
- Contact & Reservation Forms
- Live Sports Watch Area
- Smooth Animations & Page Transitions
- Custom Cursor Effects
- Animated Statistics Counter
- Image Slider & Hero Carousel
- Loading Screen Animation
- Scroll Progress Indicator

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Google Apps Script
- Google Sheets
- Font Awesome
- Google Fonts

---
